# memory-check

Use this skill to verify memory structure and recent notes.

## Checklist
- Ensure `memory/` exists.
- Ensure `memory/MEMORY.md` exists.
- Ensure today's `memory/YYYY-MM-DD.md` exists.
- Report missing files and create them if asked.
