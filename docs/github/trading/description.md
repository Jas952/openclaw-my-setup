# Trading Tools for OpenClaw

## Короткое описание для GitHub

Trading tools built for OpenClaw: market reviews, macro-calendar publishing, token optimization workflows, and Telegram-facing delivery flows.

## Расширенное описание

Этот пакет собирает наши trading-инструменты, которые проектировались для работы рядом с OpenClaw и через него: аналитические отчеты, макро-календарь, генерация визуалов, Telegram-публикации и вспомогательные сценарии.

Что важно обозначить на GitHub-странице:

- это отдельный репозиторий с trading-логикой;
- он изначально предназначен для использования вместе с OpenClaw;
- на основной репозиторий OpenClaw workspace должна вести явная ссылка;
- визуалы должны показать и результат в Telegram, и сами аналитические материалы.
