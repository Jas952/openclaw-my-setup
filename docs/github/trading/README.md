# GitHub Package: Trading Tools

Этот каталог содержит материалы для отдельной публикации `tools/trading`.

## Что подготовить

- самостоятельное позиционирование репозитория;
- явную ссылку на OpenClaw как основную среду/платформу;
- описание ключевых инструментов;
- скрины Telegram-публикаций и визуалов аналитики.

## Файлы

- `description.md` — GitHub-тексты для trading-репозитория
- `shot-list.md` — список обязательных изображений
- `assets/README.md` — структура папки под изображения
