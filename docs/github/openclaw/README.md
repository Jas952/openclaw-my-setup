# GitHub Package: OpenClaw Workspace

Этот каталог содержит материалы для оформления основного GitHub-репозитория.

## Что подготовить

- краткое описание репозитория;
- расширенное README/описание страницы;
- набор скриншотов продукта и рабочего окружения;
- скрин Telegram-канала, где используется система;
- отдельная заметка об `infrastructure/` как о внешнем источнике.

## Файлы

- `description.md` — краткий и расширенный текст для GitHub
- `shot-list.md` — список нужных визуалов
- `assets/README.md` — что именно положить в папку с изображениями
