# Shot List: OpenClaw Workspace

## Обязательные визуалы

1. Главный скрин рабочего OpenClaw-сценария.
2. Скрин интерфейса/панели управления или ключевого рабочего окна.
3. Скрин Telegram-канала, где видно, как система выглядит в реальном использовании.
4. Один общий коллаж "код + интерфейс + Telegram".

## Желательные визуалы

1. Скрин структуры workspace или отдельных инструментов.
2. Скрин одного полезного результата работы агента.
3. Обложка/hero image для GitHub README.

## Правила отбора

- не показывать токены, логины, приватные чаты и чувствительные пути;
- не использовать сырые скриншоты без ручной проверки;
- для README лучше готовить 2-3 чистых изображения, а не большую галерею.
