# OpenClaw Workspace

## Короткое описание для GitHub

Personal OpenClaw workspace with our custom tooling, trading extensions, research notes, and publication-ready integration materials.

## Расширенное описание

Этот репозиторий показывает не просто установку OpenClaw, а нашу реальную рабочую среду вокруг него: исследования, локальные инструменты, организацию workspace и прикладные сценарии использования.

Что важно обозначить на GitHub-странице:

- это наш практический workspace вокруг OpenClaw, а не официальный upstream;
- `infrastructure/` использовался как внешний базовый источник, и на оригинальный проект нужно дать ссылку отдельно;
- часть trading-наработок вынесена в самостоятельный пакет/репозиторий `tools/trading`;
- визуалы на странице должны показывать не только код, но и реальное использование: Telegram-канал, интерфейсы и итоговые материалы.
