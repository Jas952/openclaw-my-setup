# OpenClaw Workspace

Этот репозиторий собирает нашу практическую работу вокруг OpenClaw: исследование инфраструктуры, локальные расширения, рабочие инструменты и материалы для публикации.

Сейчас мы готовим workspace к публичной выкладке на GitHub в безопасном виде:

- публикуем только то, что относится к нашей собственной работе и может быть воспроизведено без локальных секретов;
- не публикуем приватные runtime-артефакты, `.env`, логи, `node_modules` и вложенные git-репозитории;
- `infrastructure/` рассматриваем как внешний источник: в публичной версии даем ссылку на оригинал и поясняем, что локально использовали его как базу;
- `tools/trading/` оформляем как отдельный репозиторий/пакет с явной ссылкой на этот workspace OpenClaw.

## Что здесь важно

- Основной аудит публикации и границы scope: `docs/github/openclaw-publishing.md`
- GitHub-материалы для основного репозитория: `docs/github/openclaw/`
- GitHub-материалы для trading-пакета: `docs/github/trading/`
- Отдельное описание trading-направления: `tools/trading/README.md`

## Статус

Это не официальный upstream OpenClaw и не зеркальная копия оригинального проекта. Это наш прикладной workspace, подготовленный к безопасной публикации и дальнейшей чистке перед первым публичным коммитом.
