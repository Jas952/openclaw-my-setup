# OpenClaw Workspace

Этот репозиторий показывает мой практический workspace вокруг OpenClaw: как был устроен бот, как была собрана UI-оболочка, как были разложены рабочие ветки и какие части системы я дорабатывал уже самостоятельно.

Важно отметить в самом начале:

- базовая работа с ботом велась на основе основного инфраструктурного репозитория OpenClaw из `infrastructure/`;
- ссылку на оригинальный upstream-репозиторий я добавлю отдельно;
- над остальными частями этого GitHub-профиля, структурой workspace, UI-слоем, прикладными сценариями и интеграциями я работал самостоятельно и в связке с языковыми моделями.

Это не официальный upstream OpenClaw и не его зеркало. Это мой прикладной слой, моя организация workspace, мои доработки и моя схема использования.

## Root Structure

Ниже кратко описана структура корневых папок этого workspace.

- `UI/` - визуальная оболочка бота, пользовательский интерфейс и связанные с ним ассеты, runtime и desktop-слой.
- `core/` - прикладные внутренние модули и сервисные компоненты, которые поддерживают отдельные функции workspace.
- `councils/` - автоматизированные проверки, отчеты, security-review логика и служебные сценарии контроля качества.
- `docs/` - материалы для GitHub, черновики описаний, структура публикации и сопроводительная документация.
- `infrastructure/` - ссылка на внешний инфраструктурный слой OpenClaw, который использовался как базовая основа.
- `skills/` - локальные skill-расширения и вспомогательные инструкции для отдельных рабочих сценариев.
- `tools/` - вспомогательные инструменты, автоматизации и изолированные утилиты вокруг основного workspace.
- `tts-jarvis/` - отдельная лаборатория и документация по голосовому/Jarvis-подобному режиму взаимодействия.
- `workspaces/` - рабочие пространства агентов, сценариев и отдельных прикладных конфигураций.
- `extracted/` - вспомогательные извлеченные материалы и внешние пакеты для локальной работы.
- `logs/` - локальные runtime-логи и служебные следы выполнения.

Ниже идет основная техническая часть про бота и его возможности.
